Тривиальные названия:
БАЗА:
CaO -> негашёная известь
Ca(OH)2 -> гашёная известь
SiO2 -> кварц, кремнозём, песок
CaSO4 * 2H2O -> гипс
NH4Cl -> нашатырь
CaCO3 -> мел, мрамор, известняк
NaHCO3 -> пищевая(питьевая) сода
Fe3O4 = FeO * Fe2O3 -> железная окалина
SiC -> карбарунд
PH3 -> фосфин

Уже не БАЗА:
KClO3 -> бертолева соль
(CuOH)2CO3 -> малахит
FeS2 -> пирит
Na3[AlF6] -> криолит
K2CO3 -> поташ
Na2CO3 * 10H2O -> кристаллическая сода
MgO -> жжёная магнезия


Ангидрид(в неорг.) - солеобразующие оксиды, проявляющие кислотные св-ва
Ангидрид(в орг.) - два кислотнчх остатка(например, карбоновых), соед. "O"

Купорос - кристаллогидрат сульфата некоторых II-валентных Me
CuSO4 * 5H2O -> медный купорос
